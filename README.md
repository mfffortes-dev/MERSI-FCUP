# MERSI-FCUP

Practical projects developed during my MSc in Network and Information Systems Engineering at FCUP.

## Repository structure

- `network-administration/` – projects and labs related to network administration
- `systems-administration/` – projects and labs related to systems administration
- `cloud-computing/` – Projects related to cloud and containers
- `network-security/` – projects and labs related to network security

## Objective

This repository organizes academic projects developed during my MSc, with a focus on practical work in networking, systems administration, cloud computing, and security.
