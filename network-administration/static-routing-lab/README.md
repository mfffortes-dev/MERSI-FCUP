# Static Routing Lab

## Description
Implementation of a network using static routing in a simulated environment.

## Technologies
- GNS3
- Linux
- Cisco IOS

## Objectives
- Configure static routes between routers
- Test connectivity between different subnets
- Simulate network failure scenarios

## Key Concepts
- Static routing
- Default gateway
- Network troubleshooting
- Packet analysis (Wireshark)

## Results
- Successful connectivity between all nodes
- Analysis of routing behavior under failure conditions
