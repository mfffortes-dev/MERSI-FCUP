## R2 (C7200)
### IP Configuration

- enable
- configure terminal

- hostname R2

- interface f0/0
    - ip address 192.168.1.130 255.255.255.192
    - no shutdown

- interface f1/0
    - ip address 192.168.50.1 255.255.255.0
    - no shutdown

- interface f2/0
    - ip address 192.168.1.193 255.255.255.192
    - no shutdown

- end
- wr

### OSPF (AS2 - Area 0)
- router ospf 2
    - router-id 2.0.0.2
    - network 192.168.50.0 0.0.0.255 area 0
    - auto-cost reference-bandwidth 1000
    **Route Redistribution and Summarization(192.168.1.0/24)**
    - redistribute ospf 1 subnets metric 100
    - summary-address 192.168.1.0 255.255.255.0

- do wr


### OSPF (AS1 - Area 0)
- router ospf 1
 - router-id 1.0.0.2
 - auto-cost reference-bandwidth 1000
 - network 192.168.1.128 0.0.0.63 area 0
 - network 192.168.1.192 0.0.0.63 area 0
 - redistribute ospf 2 subnets metric 100
 - do wr
