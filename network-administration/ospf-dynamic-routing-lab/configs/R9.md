## R9 (C7200)
### IP Configuration
- enable
- configure terminal

- hostname R9

- interface f0/0
    - ip address 172.20.1.9 255.255.255.252
    - no shutdown

- interface f1/0
    - ip address 172.17.0.1 255.255.0.0
    - no shutdown

- interface f2/0
    - ip address 172.20.1.6 255.255.255.252
    - no shutdown

- end
- wr

 - network 172.20.1.8 0.0.0.3 area 2
 - network 172.17.0.0 0.0.255.255 area 2
