## R1 (C7200)
### IP Configuration
- enable
- configure terminal

- hostname R1

- interface f0/0
    - ip address 192.168.1.129 255.255.255.192
    - no shutdown

- interface f1/0
    - ip address 192.168.1.1 255.255.255.128
    - no shutdown

- end
- wr

### OSPF - (AS1 - Area 0) 
- router ospf 1
    - router-id 1.0.0.1
    - auto-cost reference-bandwidth 1000
    - network 192.168.1.128 0.0.0.63 area 0
    - network 192.168.1.0 0.0.0.127 area 0
    - passive-interface f1/0
    - do wr
