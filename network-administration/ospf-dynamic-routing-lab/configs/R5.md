## R5 (C7200)
### IP Configuration
- enable
- configure terminal

- hostname R5

- interface f0/0
    - ip address 192.168.100.254 255.255.255.0
    - no shutdown

- end
- wr

### OSPF (AS2 - Area 0)
- router ospf 2
    - router-id 2.0.0.5

    - network 192.168.100.0 0.0.0.255 area 0
    -  auto-cost reference-bandwidth 1000

- do wr
